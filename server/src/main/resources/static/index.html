Comment démarrer
Instructions rapides

Pour utiliser cette API dans votre projet, suivez ces étapes simples :

  1. Consultez la documentation Swagger UI pour comprendre les endpoints disponibles.
  2. Référez-vous au guide d'utilisation pour des exemples concrets d'utilisation.
  3. Utilisez le format suivant pour les requêtes API : http://localhost:8080/api/{resource}
  4. Toutes les requêtes et réponses sont au format JSON.
Astuce : Si vous êtes débutant, commencez par explorer les endpoints GET qui sont sûrs et ne modifient pas les données.
